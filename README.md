# Qatar Labour Law AI Assistant

A RAG-powered legal assistant built with PyMuPDF, ChromaDB, 
Claude API, and Streamlit. Allows natural language querying 
of Qatar Labour Law with Article-level citations.


## Tech Stack
Python | ChromaDB | Claude API | Streamlit | PyMuPDF
